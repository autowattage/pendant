# Pendant
Pendant is a Hack Club YSWS (You ship We ship) program. You ship a small, decorative hardware project, and we ship a custom lanyard (quality projects will also get a custom keychain).

## local usage
1. install eleventy (`npm init -y`, `npm install @11ty/eleventy`)
2. run eleventy (`npx @11ty/eleventy`)
3. visit localhost:8080
4: profit
